Agora disponivel para Windows 10

O rastreamento de raios em tempo real para Windows 10 ultrapassa ainda mais os limites gráficos do Minecraft! Habilitado pelo mecanismo gráfico Render Dragon do Minecraft, o ray tracing traz recursos de iluminação realistas, como iluminação global e iluminação por pixel, além de suporte para texturas avançadas para o seu mundo. O rastreamento de raio está disponível para dispositivos compatíveis com DirectX, como GPU GeForce RTX e GPUs AMD selecionadas. Experimente em mundos selecionados do Marketplace, disponíveis para download gratuito!
